# Lab 2 — CSS 樣式

**學習目標：** 用 CSS 讓作品集頁面變得美觀，學習顏色、字體大小、間距的設定方式。

---

## 你要編輯的檔案

`exercise/style.css`
（`exercise/index.html` 已預先準備好，不需要修改）

---

## 如何開始

1. 用 VS Code 開啟整個 `exercise/` 資料夾
2. 右鍵 `index.html` → **Open with Live Server**
3. 開啟 `style.css`，找到每一個 `___` 並填入 CSS 值
4. 存檔後瀏覽器會立即更新
