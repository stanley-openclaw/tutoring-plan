# Lab 4 — 深色模式切換

**學習目標：** 學習 JavaScript 事件處理與 CSS class 切換，實作深色 / 淺色模式切換功能。

---

## 你要編輯的檔案

1. `exercise/index.html` — 幫按鈕加上 id（TODO 1）
2. `exercise/style.css` — 寫深色模式的 CSS（TODO 2–6）
