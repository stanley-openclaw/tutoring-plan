# Lab 1 — HTML 骨架

**學習目標：** 用 HTML 標籤建立個人作品集頁面的基本結構。

---

## 你要編輯的檔案

`exercise/index.html`

---

## 如何開始

1. 用 VS Code 開啟 `exercise/index.html`
2. 右鍵 → **Open with Live Server**（確保已安裝 Live Server 擴充套件）
3. 找到每一個 `___` 並換成你自己的內容
4. 共有 **8 個 TODO**

---

## TODO 清單

| TODO | 要做什麼 |
|------|----------|
| TODO 1 | 把分頁標題改成你的名字 |
| TODO 2 | 在 `<h1>` 裡放上你的名字 |
| TODO 3 | 寫一行簡短的自我介紹 |
| TODO 4 | 寫 2–3 句關於你自己的話 |
| TODO 5 | 加入至少 3 個你的技能 |
| TODO 6 | 寫一個專案或活動的名稱 |
| TODO 7 | 描述那個專案（1–2 句話） |
| TODO 8 | 填入你的電子郵件地址 |

---

## 完成後

查看 `ans/index.html` 對照解答。
