# Lab 5 — 個人化作品集 & 上線發布

**學習目標：** 把作品集改成你自己的內容，然後部署到 GitHub Pages，獲得一個真實可分享的網址。

---

## 你要編輯的檔案

1. `exercise/index.html` — 填入你的個人資料（TODO 1–8）
2. `exercise/script.js` — 填入你的名字和目標年份（TODO 9–10）

（`exercise/style.css` 已預先提供，不需要修改）

---

## 如何開始

1. 用 VS Code 開啟整個 `exercise/` 資料夾
2. 右鍵 `index.html` → **Open with Live Server**
3. 完成所有 10 個 TODO
4. 按照 `GUIDE.md` 的步驟把網站放到 GitHub Pages 上

---

按照 **`GUIDE.md`** 部署到 GitHub Pages，讓作品集上線！
