# Lab 3 — JavaScript 互動

**學習目標：** 用 JavaScript 讓頁面變得有互動性——學習變數、函式、DOM 操作。

---

## 你要編輯的檔案

`exercise/script.js`
（`exercise/index.html` 和 `exercise/style.css` 已預先準備好，不需要修改）

---

## 如何開始

1. 用 VS Code 開啟整個 `exercise/` 資料夾
2. 右鍵 `index.html` → **Open with Live Server**
3. 按 **F12** 開啟開發者工具，點 **Console** 分頁
4. 開啟 `script.js`，找到每一個 `___` 並填入正確的值或程式碼
